About

Housing Sales Trend Analysis

Project Overview

The Housing Sales Trend Analysis project is developed using Tableau Public to analyze residential housing data. The dashboard provides valuable insights into house prices, sales performance, location-based trends, bedroom distribution, and yearly growth using interactive visualizations.

  • Interactive Dashboard
  • Location Wise House Analysis
  • Price Distribution
  • Story Based Insights
  • Dynamic Filters

This project helps users understand housing market trends through rich data visualizations built using Tableau Public.

Dashboard

Comprehensive Housing Sales Dashboard

Dashboard Highlights

The interactive dashboard presents a complete overview of housing sales trends using Tableau Public. It allows users to explore pricing trends, location-based analysis, house condition, bedrooms, bathrooms, living area and yearly sales.

  • Interactive Filters
  • House Price Analysis
  • Location Wise Sales
  • Bedrooms & Bathrooms
  • Dynamic Tableau Dashboard

Story

Housing Sales Story Analysis

Story Insights

The Tableau Story presents the complete workflow of the Housing Sales Analysis. It explains how the data was analyzed using different dashboards and visualizations to discover trends, relationships, and business insights.

  • Interactive Story Navigation
  • Data-driven Insights
  • Trend Analysis
  • Sales Performance
  • Final Business Conclusions

The Story combines multiple dashboard views into a single presentation, making it easier to understand the housing sales data and key findings.
