# Dataset

## Overview

The Housing Market Analysis project uses a transformed housing dataset in **CSV (Comma-Separated Values)** format. The dataset contains residential property information that is used to analyze housing market trends and create interactive visualizations using Tableau Desktop Public.

---

## Dataset Description

The dataset contains various attributes related to residential properties, including property prices, structural characteristics, renovation details, and location information. It was used to perform data analysis, create visualizations, design dashboards, and develop storyboards.

During the preprocessing stage, duplicate records were identified and removed to improve data quality. The cleaned dataset was then imported into Tableau Desktop Public for further analysis.

## Data Preparation

The following preprocessing activities were performed before visualization:

- Imported the dataset into Tableau Desktop Public.
- Verified field names and data types.
- Identified and removed duplicate records.
- Validated the dataset for consistency.
- Created calculated fields such as **House Age** and **Years Since Renovation**.
- Prepared the dataset for dashboard and storyboard development.
